# Feedback

A running log of lessons that steer synthesis. Every line here is read on each run, so keep
it short and imperative — "do X", "stop doing Y".

Two ways entries land here:

1. **You write them.** Reacted to a note and want that to stick? Add a line.
2. **The app proposes them.** When it detects that you hand-edited a note, it infers the
   preference behind your edit and appends a suggestion under `## Proposed`. Promote a
   suggestion by moving it above the line; delete the ones you don't want.

## Active lessons

- Keep summaries tight — one or two sentences before the first heading, not a paragraph.
- Preserve my parenthetical asides; don't sand them out.
- When a concept deserves its own note but doesn't have one yet, link it anyway (a stub
  `[[wikilink]]`) instead of skipping the link or writing it out in prose.
